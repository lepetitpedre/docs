## Bulk updates
Yes bulk updates are available on almost all end-points, try to limit to 2000 objects per update. 

## If on object fails in the request fails, the complete request fails
If the request fails, the compelte request have failed. So if you do an update, and it failed, non of the objects where updated in the request. 

## Number of requests
Try to limit the number of requests to 2 per minute
