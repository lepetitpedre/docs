### Username and password

From the PriceEdge team or you should have received a username. The password, if not already set, you set via the link in mail you should have received. 


<Note>In this example the username is pricing@example.com and **_MYPASSWORD123_ as the password</Note>

### API url
The`{api_url}` is **your company specific url** + priceedge.eu/papi/api. 

If your company url is **yourdomain** then the API-url would be `yourdomain.priceedge.eu/papi/api`


<Note>In this example the API-url is  is **yourdomain.priceedge.eu/papi/api**</Note>



### First request - GET item strucutre 
<Note>Don' froget to change to you domain, username and password</Note>

```http HTTP Example
GET /papi/api/tables/item
Host: yourdomain.priceedge.eu
Authorization: ApiKey pricing@example.com:MYPASSWORD123
Content-Type: application/json
```
```bash cURL Example
curl 'https://yourdomain.priceedge.eu/papi/api/tables/item' \
--header 'Authorization: ApiKey pricing@example.com:MYPASSWORD123' \
--header 'Content-Type: application/json'
```

<Warning>ApiKey is case sensitive and note the : between username and password</Warning>
