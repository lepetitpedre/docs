### Price Objects

Price Objects are one of the core parts of the Price Edge data structe. 
  It's your items, customers, pricelists and so on. Every main price object exist in price edge out of the box. On top of the price objects every price object has _Extensions_. 
An extension is an associated table that holds addiational attributes that are linekd to the main price object. 



| Price Object     | Description | Extension |
| ----------- | ----------- | ----------- |
| Item      | Main object to hold you items      | Item_Extension
| Pricelists   | Your Pricelists      |
| Customers   |       |
| Price Groups   |        |
| Competitors  |        |
